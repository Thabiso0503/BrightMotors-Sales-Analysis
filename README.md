###🚗 Bright Motors Car Sales Analysis
##📌 Problem Statement

# Bright Motors has recently appointed a new Head of Sales tasked with improving overall sales performance, expanding dealership reach, and optimizing inventory. However, the company currently lacks clear, data-driven insights into which vehicles drive revenue, how pricing factors influence sales, and which regions perform best.

# This project addresses the need to transform raw transactional car sales data into actionable insights that support strategic decision-making.

# 🎯 Aim of the Project

#  The aim of this project is to analyze historical car sales data to uncover key trends, patterns, and performance drivers that will help Bright Motors improve profitability, enhance sales strategies, and make informed business decisions.

# Objectives

# To achieve the project aim, the following steps were taken:

# 📊 Data Understanding & Planning
Designed a data architecture and workflow using Miro
Identified key business questions and key performance metrics
# 🧹 Data Preparation & Processing
Converted raw dataset into a structured format (CSV)
Cleaned data (handled missing values, removed duplicates, standardized formats)
Transformed data using SQL (Databricks)
Created calculated fields such as:
Total Revenue
Profit Margin
Sales groupings by time and region
#🔍 Data Analysis
Queried the dataset using SQL to extract insights
Analyzed relationships between price, mileage, and year
Evaluated top-performing car makes, models, and regions
# 📈 Data Visualization
Built dashboards and visualizations using Excel and Power BI
Added filters (region, fuel type, year) for interactivity
# 🧾 Presentation & Reporting
Created a professional presentation using Canva
Summarized key insights and recommendations for stakeholders
# 📊 Summary of Results

# The analysis revealed several key insights:

Certain car makes and models contribute significantly more to total revenue
Sales performance varies across regions, highlighting high-performing locations
Pricing, mileage, and manufacturing year strongly influence purchasing behavior
Customer preferences show trends toward specific fuel types and vehicle categories
Opportunities exist to improve profitability through better inventory and pricing strategies
# 🛠️ Tools Used
SQL (Databricks) – Data querying and transformation
Microsoft Excel – Data visualization and charts
Power BI – Interactive dashboards
Miro – Project planning and data architecture
Canva – Presentation design
